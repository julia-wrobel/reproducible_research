About This Website

This is an R Markdown website for a reproducible research workshop at SISMID 2024.

In the short course, we’ll use several packages which are listed below. Code to install the packages follows, and version information about each of these packages and their dependencies is at the bottom of this page.

Installing packages from CRAN

# main packages
install.packages(c("tidyverse"))

# additional packages used for visualization
install.packages(c("viridis", "patchwork"))

Installing packages from GitHub

# install projectr from GitHub
install.packages("devtools")
devtools::install_github("julia-wrobel/projectr")
